# slide-generator

## Overview

Converts pasted content, markdown text, or screenshot descriptions into HTML presentation slides using the CreateFuture snippet system. Runs end-to-end automatically without step-by-step setup questions.

## When to use it

- You want to turn pasted text, notes, or bullet points into presentation slides.
- You need to generate a deck from a screenshot description or a block of markdown.
- You want slides built without manual setup or prompting.

## Example prompts

- "Create slides based on this: *(paste content)*"
- "Turn this into a presentation."
- "Make slides from this content."
