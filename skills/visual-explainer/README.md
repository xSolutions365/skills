# visual-explainer

## Overview

Generate a self-contained HTML explainer for technical systems, plans, or structured data with clear visual hierarchy and browser-friendly output. This prompt is based on an opinionated adaptation of [nicobailon/visual-explainer](https://github.com/nicobailon/visual-explainer) and incorporates ideas from [cyxzdev/Uncodixfy](https://github.com/cyxzdev/Uncodixfy/tree/main) to improve Codex output quality and avoid generic UI patterns.

## When to use it

- You want a visual explainer instead of an ASCII diagram or plain text summary.
- You need a single HTML artifact that can be opened and reviewed in a browser.
- You want technical content rendered as diagrams, tables, slides, or structured layouts.

## Example prompts

- "Use `visual-explainer` to create an HTML overview of this system architecture."
- "Use `visual-explainer` to turn this rollout plan into a browser-based visual summary."
- "Use `visual-explainer` to render these metrics as a readable HTML table with commentary."
