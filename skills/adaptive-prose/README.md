# adaptive-prose

## Overview

Draft or redraft prose with an explicit writing preset and a review pass that checks for clarity, audience fit, and meaning preservation.

## When to use it

- You need new prose for a defined audience or purpose.
- You need to improve existing prose without changing its intent.
- You want the writing style to be chosen explicitly, not implied.

## Example prompts

- "Use `adaptive-prose` with the `operator-brief` preset to turn these notes into a client update."
- "Use `adaptive-prose` with the `surgical-redraft` preset to tighten this memo without changing any claims."
- "Use `adaptive-prose` with the `narrative-explainer` preset to rewrite this technical summary for a broader audience."
