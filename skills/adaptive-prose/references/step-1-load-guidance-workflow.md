# Step 1 Workflow: Load universal and preset guidance

## Objective

Translate the chosen preset into a concrete drafting brief for the current run.

## Required actions

1. Read [preset-selection-guide.md](preset-selection-guide.md).
2. Read [prose-style-guide.md](prose-style-guide.md).
3. Read the guide that matches the selected `--preset`.
4. If the run is a redraft, read [redraft-integrity-guide.md](redraft-integrity-guide.md).
5. Extract a live drafting brief for this run with:
   - primary reader outcome,
   - tone boundary,
   - opening rule,
   - structure pattern,
   - allowed layout shapes,
   - claim-strength rule,
   - target word range or compression target,
   - verbosity ceiling,
   - ornamentation tolerance,
   - banned phrase set,
   - closing rule,
   - redraft preservation rule when applicable.
6. If the current preset is visibly wrong for the task, stop and reselect instead of blending two presets together.

## Done when

- The drafter can state the selected preset's job in one sentence.
- The current run has a concrete drafting brief derived from the references.
- Redraft preservation rules are active when source text must be preserved.
