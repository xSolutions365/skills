# Preset Guide: Surgical Redraft

## Objective

Define how to strengthen existing prose when the meaning is already sound and the main job is clarity, rhythm, and economy.

## Required actions

1. Preserve the source's meaning, scope, and level of commitment.
2. Tighten vague or padded phrasing without changing what is being claimed.
3. Improve sentence flow, paragraph order, and emphasis only where clarity demands it.
4. Keep close to the source structure unless the source structure actively obscures the point.
5. Default to a lighter, shorter result than the source when possible, usually reducing visible padding by roughly 10-20% without losing necessary content.
6. Run a factual parity pass before accepting the rewrite.

## Done when

- The rewrite reads cleaner than the source.
- Meaning and commitment remain intact.
- Structural changes are justified by clarity rather than preference.
- The revision tightens the original instead of silently expanding it.
