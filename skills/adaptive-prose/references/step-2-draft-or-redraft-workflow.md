# Step 2 Workflow: Draft or redraft the prose

## Objective

Produce one coherent draft that matches the selected preset, stays bound to the supplied context, and reads as intentional prose rather than assistant filler.

## Required actions

1. Start from the user's actual situation, source text, or working context rather than a generic template opening.
2. Shape the draft according to the selected preset:
   - `operator-brief`: answer first, then minimum necessary support.
   - `technical-deep-dive`: conclusion, assumptions, reasoning, implications.
   - `narrative-explainer`: premise, explanation, consequence.
   - `executive-decision-brief`: situation, implication, recommendation, risk or next move.
   - `surgical-redraft`: preserve structure unless it blocks clarity.
   - `voice-shift-redraft`: allow structural rewrite when audience fit requires it.
3. Bind every major claim to supplied context, source text, or explicit user instruction.
4. Honor the preset's length discipline and layout shape while drafting. If the draft is drifting long or defaulting to same-shaped paragraphs, compress or restructure before polishing sentences.
5. Remove generic filler while drafting rather than polishing it later.
6. Keep paragraph purpose distinct so the argument moves forward instead of repeating itself.
7. Use a short flat list only when there are genuinely sibling points such as decisions, risks, failure modes, or next steps. Do not use lists as a substitute for argument.
8. End in a way that fits the preset:
   - with an action or blocker for `operator-brief`,
   - with an implication for `technical-deep-dive`,
   - with clearer orientation for `narrative-explainer`,
   - with a decision or next-step implication for `executive-decision-brief`.

## Done when

- The draft matches the selected preset.
- The prose reads as one deliberate piece rather than a sequence of fragments.
- Every major claim is grounded in supplied material.
- The length and layout would let a reader distinguish this preset from a neighboring one.
