# Workflow

### Step 0: Apply human writing guidance

- **Purpose**: Produce or revise writing so it reads like knowledgeable human prose rather than AI-generated text.
- **When**: Run for any writing task where the user asks for human-sounding prose, lower AI-detection risk, or removal of AI style markers.
- Follow the reference exactly, including the critical violations, mandatory verification checks, vocabulary replacements, phrase cuts, structural patterns, tone guidance, and formatting rules.
- Preserve the user's intended meaning, claims, and audience while removing AI-detectable vocabulary and structure.
- Workflow: [references/human-writing-style-workflow.md](references/human-writing-style-workflow.md)

## Output

### Result Format

- Return the finished prose in the format requested by the user.
- If revising existing text, preserve factual content and state only material changes when useful.
- Do not include a separate verification report unless the user asks for one.
