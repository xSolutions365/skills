# human-writing-style

## Overview

Produces writing that sounds authentically human by removing AI-detectable vocabulary, sentence patterns, formatting habits, and promotional phrasing while preserving the user's intended meaning.

## When to use it

- You need prose that should not read like AI-generated content.
- You are drafting or revising articles, reports, blog posts, emails, creative writing, proposals, documentation, or professional communications.
- You want a mandatory verification pass for flagged vocabulary, em dashes, repeated sentence starts, uniform structure, and press-release tone.

## Example prompts

- "Use `human-writing-style` to rewrite this blog post so it sounds less AI-generated."
- "Use `human-writing-style` to draft this email in a natural professional voice."
- "Use `human-writing-style` to remove AI tells from this proposal without changing the claims."
